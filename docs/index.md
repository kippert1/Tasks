# Сборник задач

Значение арифметического выражения $6^{260}+6^{160}+6^{60}-x$, где $x$ — целое положительное число, не превышающее $2030$, записали в $6$-ричной системе счисления.

Определите наименьшее значение $x$, при котором количество нулей в $6$-ричной записи равно $202$.

**Ответ:** в десятичной системе счисления.
